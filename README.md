# MovieDB-Android-Tracker-Demo
An Android App that fetches Popular Movies from [The Movie Database API](https://developers.themoviedb.org) using

  - Kotlin
  - MVVM Pattern
  - Android Architecture Components (LiveData, Room, ViewModel)
  - RecyclerView
  - Retrofit
  - Glide
  - Gson
  - Moshi
